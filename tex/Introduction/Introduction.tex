\subsection{Benefits of Scout}

\noindent Existing Documentation
\begin{itemize}
  \item concept wiki \url{http://wiki.eclipse.org/Scout/Overview/Why_You_Should_Use_Scout}
\end{itemize}

Eclipse Scout is a mature and open framework for modern, 
service oriented business applications.

With its multi-frontend support, Scout applications may run as desktop 
applications, in the web or on a mobile using a single codebase.

Scout substantially boosts developer productivity and is simple to learn.

User friendly applications are straight forward to implement with Scout’s 
comprehensive set of user interface components.

Completely based on Java/Eclipse, Scout Applications are easy to integrate 
in most IT environments.

\section{What should I read?} 
\seclabel{whatshouldiread}

The text below provides guidelines on what to read (or what to skip) depending on your existing background.

We first address the needs of junior Java developers that like to learn more about developing enterprise applications.
Then, we suggest a list of sections relevant for software wizards that already have a solid understanding of the Eclipse platform, Java enterprise technolgies, and real world applications.
Finally, the information needs of IT managers are considered.

\subsection{I know some Java}

The good news first.
This book is written for you! 
No prior knowledge regarding the \href{http://wiki.eclipse.org/Platform}{Eclipse Platform} is needed. 
Of course, having used the Eclipse IDE for Java development is helpful - please do not mistake the Eclipse IDE with the Platform.
We do not even assume that you have a meaningful understanding of the 
\href{http://en.wikipedia.org/wiki/Java_Platform,_Enterprise_Edition}{Java enterprise edition}.

The bad news is, that writing Scout applications requires you to have a solid understanding of 
\href{http://en.wikipedia.org/wiki/Java_%28software_platform%29}{Java}.
To properly benefit from this book, we assume that you have been developing software for at least a year.
In addition you should master the 
\href{http://en.wikipedia.org/wiki/Java_SE}{Java standard edition} 
to a significant extent. 
To be more explicit, you are expected to be comfortable with all material required for the 
\href{http://docs.oracle.com/javase/tutorial/extra/certification/javase-7-programmer1.html}{Programmer Level I Exam}
and most of the material required for the
\href{http://docs.oracle.com/javase/tutorial/extra/certification/javase-7-programmer2.html}{Programmer Level II Exam}.

In this book we try to provide all necessary material that will help you to successfully build high quality enterprise applications.
